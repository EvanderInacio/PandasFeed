## 📝 Descrição 

Pandas Feed é um website para gerenciar suas postagens. 

- Website: [pandas-feed.vercel.app](https://pandas-feed.vercel.app/)

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:
